# Email-шифратор (симетричне шифрування, Python)

Ця програма демонструє роботу **симетричної криптографії** та моделює механізм **захищеної електронної комунікації**.  
Реалізовано простий варіант шифрування текстових повідомлень із використанням алгоритму Fernet (AES + HMAC) та генерацією ключів на основі персональних даних користувача.

Програма повністю консольна, підтримує експорт/імпорт ключів у вигляді файлів, збереження зашифрованих повідомлень та не потребує ручного введення чи копіювання шифртексту.

## Функціонал

* Генерація симетричного ключа на основі персональних даних:
  * email користувача;
  * секретна фраза (може містити прізвище, рік народження тощо);
  * ключ формується як: `key = Base64( SHA256(email + secret_phrase) )`.
* Відображення поточного активного ключа у форматі Base64.
* Експорт ключа у файл (наприклад, `mykey.key`) для передачі іншому користувачеві.
* Імпорт ключа з файлу, отриманого від іншої особи.
* Шифрування текстових повідомлень за допомогою алгоритму Fernet:
  * введення відкритого тексту;
  * шифрування в один рядок шифртексту;
  * можливість збереження зашифрованого повідомлення у файл (наприклад, `message.txt`).
* Розшифрування повідомлень із файлу:
  * завантаження шифртексту з файлу;
  * розшифрування за допомогою активного ключа;
  * виведення відновленого тексту у консоль.
* Обробка помилок:
  * відсутній ключ;
  * невірний шлях до файлу;
  * спроба розшифрувати повідомлення «чужим» ключем.

## Вимоги

* Python 3.x  
* Бібліотека `cryptography` (модуль `Fernet`)  
* Стандартні бібліотеки (`base64`, `hashlib`, `os`)

> Встановлення додаткового пакета:
> ```bash
> pip install cryptography
> ```

## Встановлення
1. Переконайтесь, що у вас встановлено Python 3.x  
   Завантажити можна з [офіційного сайту](https://www.python.org/downloads/).
2. Встановіть бібліотеку `cryptography`:
   ```bash
   pip install cryptography
3. Запустіть програму:
    ```bash
    python email_encryptor.py
    ```
4. Дотримуйтесь інструкцій у консолі.
