# cybersecurity-labs-malysheva
# Лабораторні роботи з предмету «Захист інформації»

Репозиторій містить виконані лабораторні роботи студентки **Малишевої Юлії**, групи **6.04.122.010.22.1**.

## Структура репозиторію

Кожна лабораторна робота знаходиться в окремій директорії (`lab01`, `lab02` і т.д.), що містить:
* Файл з кодом програми (`.py`).
* Файл `README.md` з описом завдання, функціоналу та інструкцією запуску.
* Папку `screenshots` з прикладами роботи (де це необхідно).

---

## Зміст лабораторних робіт

### [➡️ Лабораторна робота №1: Аудит власної цифрової безпеки](./lab01/)
* **Опис:** Аудит безпеки пошти, аналіз надійності паролів, налаштування приватності в соціальних мережах та розробка програми-аналізатора безпеки паролів на Python.
* **Програма:** `password_checker.py` - консольна утиліта для оцінки складності пароля.

### [➡️ Лабораторна робота №2: Системи захищеного листування та класичне шифрування](./lab02/)
* **Опис:** Дослідження класичних шифрів (Цезар, Віженер), проведення криптоаналізу (brute force) та розробка програми для порівняння шифрів Цезаря та Віженера з підтримкою українського алфавіту.
* **Програма:** `cipher_comparison.py` - консольна утиліта для шифрування/розшифрування тексту.

### [➡️ Лабораторна робота №3: Стеганографія у зображеннях (LSB-метод)](./lab03/)
* **Опис:** Реалізація приховування та витягування текстових повідомлень у зображеннях методом LSB, опційне XOR-шифрування, формування службового заголовка, побітове вбудовування даних та аналіз змін у стего-файлі (diff-карта, відсоток змінених пікселів).
* **Програма:** `stego.py` – консольна утиліта для приховування та витягування даних із зображень.

### [➡️ Лабораторна робота №4: Асиметричне шифрування та цифровий підпис (RSA)](./lab04/)
* **Опис:** Реалізація спрощеної системи електронного цифрового підпису (ЕЦП) на основі RSA-подібного алгоритму: генерація ключів, створення підпису, перевірка достовірності, автоматичне керування кількома ключами, виявлення змін у файлах та побудова консольного інтерфейсу з меню.
* **Програма:** `digital_signature_system.py` – консольна утиліта для створення, збереження та перевірки цифрових підписів із підтримкою менеджера ключів.

### [➡️ Лабораторна робота №5: Симетричне шифрування електронних повідомлень](./lab05/)
* **Опис:** Реалізація симетричного алгоритму шифрування електронних повідомлень із використанням криптографічної схеми Fernet (AES + HMAC). Програма демонструє механізм захищеної комунікації: генерацію ключів на основі персональних даних, експорт та імпорт ключів, шифрування текстових повідомлень та їх розшифрування на стороні отримувача.
* **Програма:** `email_encryptor.py` – консольна утиліта для створення симетричних ключів, шифрування повідомлень і розшифрування зашифрованих файлів.

### [➡️ Лабораторна робота №6: SQL-інʼєкції та методи захисту](./lab06/)
* **Опис:** Дослідження вразливостей типу SQL Injection на основі консольного застосунку з базою даних SQLite. Реалізовано дві версії роботи із SQL-запитами: вразливу (з прямим підставленням введення користувача) та захищену (на prepared statements). Програма моделює обхід авторизації, витяг даних через інʼєкцію у пошук, а також демонструє, як параметризація запитів повністю блокує інʼєкції.
* **Програма:** `sql_demo.py` – консольна утиліта з меню, яка порівнює роботу вразливих та захищених SQL-запитів, демонструє типові пейлоади та результати інʼєкцій.

### [➡️ Лабораторна робота №7: Комплексний двоетапний захист файлів](./lab07/)
* **Опис:** Реалізація комплексної системи захисту, що поєднує два криптографічні методи: симетричне шифрування файлу алгоритмом AES-256 та приховування зашифрованих даних у зображенні за допомогою LSB-стеганографії. Програма виконує повний цикл «захист → відновлення», забезпечує перевірку цілісності SHA-256, збір метрик ефективності (час, розмір до/після) та формує аналітичний звіт.
* **Програма:** `complex_protection.py` – консольна утиліта, яка шифрує файл, приховує його в зображенні, відновлює початкові дані та автоматично аналізує ефективність роботи системи.
