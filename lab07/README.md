# Complex Protection (AES шифрування + LSB стеганографія, Python)

Ця програма демонструє роботу **двоетапного захисту файлів** та моделює механізм комбінування симетричного шифрування й стеганографії.  
Реалізовано простий приклад застосунку, у якому спочатку виконується шифрування файлу алгоритмом AES-256, а потім зашифровані дані приховуються всередині зображення за допомогою LSB-стеганографії.

Програма повністю консольна, автоматично працює з обраними користувачем файлами, підтримує повний цикл «захист → відновлення» та не потребує складних налаштувань.

## Функціонал

* Генерація сеансового AES-ключа на основі персональних даних користувача (ПІБ, дата народження, секретна фраза).
* Двоетапний захист файлу:
  * симетричне шифрування вмісту файлу за алгоритмом AES-256 у режимі CBC;
  * приховування зашифрованих даних у зображенні методом LSB-стеганографії (формування стегоконтейнера у форматі PNG).
* Повний цикл відновлення:
  * витягування payload зі стегозображення;
  * розшифрування даних тим самим сеансовим ключем;
  * відновлення початкового файла.
* Перевірка цілісності:
  * обчислення SHA-256 хеша оригінального файлу;
  * збереження хеша разом із зашифрованими даними;
  * порівняння хеша при відновленні для виявлення змін або помилок ключа.
* Аналітичний модуль:
  * фіксація розміру даних до та після кожного етапу;
  * вимірювання часу виконання шифрування, стеганографії, витягування та розшифрування;
  * виведення зведеного аналітичного звіту в табличному вигляді.
* Обробка базових помилок:
  * відсутність вказаних файлів;
  * некоректний формат стегоконтейнера;
  * неможливість розшифрування при використанні іншого сеансу або неправильного ключа.

## Вимоги

* Python 3.x  
* Бібліотека Pillow (`PIL`)  
* Бібліотека PyCryptodome (`Crypto`)  

## Встановлення
1. Переконайтесь, що у вас встановлено Python 3.x  
   Завантажити можна з [офіційного сайту](https://www.python.org/downloads/).
2. Встановіть необхідні бібліотеки:
    ```bash
    pip install pillow pycryptodome
    ```
3. Завантажте файл програми `complex_protection.py`.
4. Запустіть програму:
    ```bash
    python complex_protection.py
    ```
5. Дотримуйтесь інструкцій у консолі.
