# Система цифрових підписів (RSA-подібна, Python)

Ця програма демонструє роботу **асиметричної криптографії** та механізму **електронного цифрового підпису (ЕЦП)**.  
Реалізовано спрощений варіант RSA з можливістю створення підписів, перевірки їх достовірності та керування кількома ключами.

Програма повністю консольна, підтримує автоматичне збереження ключів у `keys.json`, автоматичне формування `.sig` файлів і не потребує ручного введення підписів.

## Функціонал

* Генерація ключів RSA-подібним методом на основі персональних даних користувача.
* Збереження та завантаження кількох ключів у `keys.json`.
* Вибір активного ключа для роботи.
* Створення цифрового підпису для будь-якого файлу:
  * обчислення SHA-256
  * приведення хешу за модулем `n`
  * підпис: `signature = hash^d mod n`
* Автоматичне створення файлу підпису `<filename>.sig`.
* Перевірка підпису публічним ключем:
  * `verified_hash = signature^e mod n`
  * порівняння з актуальним хешем файлу
* Повне виявлення підробок та змін у файлі.
* Видалення ключів з менеджера.
* Обробка помилок: неправильний шлях, відсутній підпис, відсутні ключі тощо.

## Вимоги

* Python 3.x  
* Стандартні бібліотеки (hashlib, json, os, random, math)
  
> Додаткові пакети **не потрібні**.

## Встановлення

1. Переконайтесь, що у вас встановлено Python 3.x  
   Завантажити можна з [офіційного сайту](https://www.python.org/downloads/).
2. Завантажте файл `digital_signature_system.py`.
3. Помістіть файл у будь-яку директорію.
4. (Необов’язково) Видаліть старий `keys.json`, якщо він з попередніх запусків.

## Використання

1. Відкрийте термінал або IDE (VS Code, PyCharm).
2. Перейдіть у директорію з файлом:
    ```bash
    cd path/to/directory
    ```
3. Запустіть програму:
    ```bash
    python digital_signature_system.py
    ```
4. Дотримуйтесь інструкцій у консолі.
