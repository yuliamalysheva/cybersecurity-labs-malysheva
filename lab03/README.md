# Стеганографія в зображеннях (LSB-метод)

Ця програма демонструє приховування та витягування текстової інформації в зображеннях за допомогою методу **LSB-стеганографії**. Вона формує службовий заголовок (маркер, довжина, payload), виконує вбудовування бітів у молодші біти пікселів, підтримує опційне XOR-шифрування та надає базовий аналіз змін у зображенні.

## Функціонал

* Реалізація функцій приховування та витягування даних методом **LSB**.
* Формування службового блоку (4-байтний маркер STEG, довжина повідомлення, payload).
* Підтримка текстових повідомлень у кодуванні UTF-8.
* Опційне **XOR-шифрування** повідомлення перед вбудовуванням.
* Генерація diff-карти та визначення відсотка змінених пікселів.
* Порівняння розміру оригінального та модифікованого зображення.
* Обробка помилок введення (некоректний файл, неправильний режим, відсутні приховані дані).

## Вимоги

* Python 3.x
* Бібліотека Pillow (PIL)

## Встановлення

1.  Переконайтеся, що у вас встановлено Python 3.x. Завантажити можна з [офіційного сайту](https://www.python.org/downloads/).
2.  Завантажте файл `stego.py` із цього репозиторію.
3.  Встановіть бібліотеку Pillow командою:
    ```bash
    pip install pillow
    ```

## Використання

1.  Відкрийте термінал (командний рядок) або IDE (наприклад, PyCharm, VS Code).
2.  Перейдіть до директорії з файлом `stego.py`.
3.  Запустіть програму командою:
    ```bash
    python stego.py
    ```
4.  Дотримуйтесь інструкцій у консолі для вибору режиму, введення повідомлення, ключа та файлів.
